# CodeSieve

Grade code through 7 principle-based sieves (KISS, DRY, SRP, Naming, Nesting, Complexity, Comments).

## Install

```bash
pip install codesieve
```

## Usage

```bash
codesieve scan file.py
codesieve scan src/ --format json
codesieve scan . --deterministic --fail-under 6
codesieve sieves
```
